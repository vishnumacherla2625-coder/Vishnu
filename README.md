# Babai Photo Studio Website

A professional portfolio website for Babai Photo Studio showcasing photography services and gallery.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations** - Fade-in and scroll animations for better UX
- **Gallery Lightbox** - Click on gallery images to view in full screen
- **WhatsApp Integration** - Quick contact button linked to WhatsApp
- **Modern Styling** - Clean, professional design with hover effects
- **SEO Optimized** - Proper meta tags and semantic HTML

## Structure

```
├── index.html      - Main HTML file
├── styles.css      - Styling and responsive design
├── script.js       - JavaScript for interactivity
└── README.md       - This file
```

## How to Customize

1. **Update Contact Information**
   - Replace phone number in contact section and WhatsApp links
   - Update email address
   - Add studio address

2. **Update Gallery Images**
   - Replace image URLs in the gallery section with your own photos

3. **Change Colors**
   - Primary color: `#ff9800` (orange)
   - Background: `#111` (dark)
   - Update these in `styles.css`

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize with your studio information
4. Deploy to your web hosting
